## How to run API
### setup virtual environment
- `python3 -m virtualenv .venv`
- `source .venv/bin/activate`

### install requirements
- `pip install -Ur requirements.txt`

### run application
- `python src/app.py`

### run test cases
- `pytest tests/`


## Assumptions:
1. Multiple ingredients can be toppedup at a time.
